Form-related utilities and wrappers for consistent task input surfaces.

## Guidelines

- Always label inputs (visually or via `aria-label`).
- Prefer composition over large monolithic form components.
- Keep visual states (error, success) aligned with daisyUI semantic colors.

## Adding a Form Component
1. Implement under `src/components/forms/`.
2. Create `Component.stories.tsx` with:
```ts
const meta: Meta<typeof Component> = {
	title: 'Forms/Component',
	component: Component,
	parameters: { docs: { description: { component: 'Short description.' } } },
	tags: ['autodocs']
};
export default meta;
```
3. Document validation expectations succinctly in the description or additional stories.
