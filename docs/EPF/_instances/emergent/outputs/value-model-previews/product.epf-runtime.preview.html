Product Mission

Enable AI agents, DevOps pipelines, and power users to execute EPF methodology cycles programmatically with durable workflows, schema validation, and knowledge graph integration.

Overview

EPF-Runtime is a headless workflow orchestration system enabling AI agents and humans to execute EPF methodology cycles programmatically. Provides durable workflow execution, schema validation, and deep integration with Emergent Core knowledge graph. Shares auth/RLS infrastructure with Core.

Runtime as OS Layer

EPF-Runtime is designed to evolve into the "OS layer" for the Emergent ecosystem, where Emergent Core runs as an application on the Runtime platform. This architecture will enable centralized auth/authz across all Emergent apps, shared workflow orchestration, an event bus for app-to-app communication, and platform APIs for third-party integrations.
